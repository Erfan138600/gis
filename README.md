# سیستم اطلاعات جغرافیایی آموزشی فارسی (Persian Educational GIS)

یک نرم‌افزار **کاملاً فارسی**، زیبا، حرفه‌ای و آموزشی برای یادگیری و تمرین مفاهیم **سیستم اطلاعات جغرافیایی (GIS)** با زبان پایتون.

مناسب برای:
- دانشجویان جغرافیا و GIS
- اساتید دانشگاه
- دانش‌آموزان دبیرستان (پایه دوازدهم رشته انسانی)
- علاقه‌مندان به برنامه‌نویسی جغرافیایی

## ویژگی‌های کلیدی

- رابط گرافیکی فوق‌العاده زیبا و مدرن (شبیه نرم‌افزارهای حرفه‌ای مثل QGIS)
- کاملاً فارسی با فونت‌های خوانا
- سه بخش مجزا: **ورودی ← پردازش ← خروجی**
- پشتیبانی از فرمت‌های استاندارد:
  - Shapefile (.shp)
  - GeoJSON
  - KML / KMZ (گوگل ارث)
  - CSV با مختصات (lat/lon)
- ابزارهای واقعی پردازش جغرافیایی:
  - ایجاد بافر (Buffer)
  - محاسبه مساحت و محیط (به مترمربع و هکتار)
  - اتحاد هندسی (Union)
  - تقاطع (Intersection) ← در نسخه‌های بعدی
- نمایش نقشه تعاملی با **Folium** (قابلیت زوم، رسم، اندازه‌گیری فاصله و مساحت)
- ذخیره خروجی به صورت Shapefile و GeoJSON
- تم تیره و روشن (Dark/Light Mode)
- مناسب برای ارائه در کلاس، کارگاه و پروژه دانشجویی
